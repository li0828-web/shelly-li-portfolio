# Portfolio Website - Shelly Li

A modern portfolio website built with React and Docker.


- **Shelly Li**
- GitHub: https://github.com/li0828-web


## Technologies Used
- React 18
- React Router DOM 6
- CSS3
- Docker
- Nginx

## Features
- Responsive Design
- Multi-page navigation
- Docker containerized
- ESLint and Prettier for code quality

## Getting Started

### Development Mode
1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open http://localhost:3000


## Portfolio Sections
1. Basic Information - About me and contact details
2. Work - Featured projects with tech stacks
3. Skills - Languages, frameworks, and tools
4. Resources - Learning materials and documentation
5. Developer Setup - VS Code and terminal configuration

## Available Scripts
- `npm start` - Run development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
